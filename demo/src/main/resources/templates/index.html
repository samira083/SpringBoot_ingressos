Sistema de Gestão de Ingressos

Interface para solicitação e gerenciamento de ingressos de eventos culturais.

Solicitar Ingressos

O sistema utiliza os dados do comprador para registrar a emissão dos ingressos.

Painel de Emissão

Após informar seus dados, o sistema registra o comprador e permite consultar todos os usuários cadastrados.

Resumo da Operação

  • Nome do comprador
  • Gmail cadastrado
  • Evento selecionado
  • Quantidade de ingressos
  • Confirmação da emissão
